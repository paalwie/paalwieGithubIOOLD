Projektliste

  • Netzplan-Tool Link zu Netzplan-Tool

    Dieses Java-Programm ermöglicht die Berechnung und Analyse von Netzplänen, wie sie in Bauprojekten, der industriellen Fertigung oder anderen komplexen Vorhaben eingesetzt werden. Es unterstützt verschiedene Netzplantechniken, darunter die Meilensteinmethode (Milestone Method), die Critical Path Method (CPM) und die Berechnung von EARLIEST START (ES), LATEST START (LS), EARLIEST FINISH (EF), LATEST FINISH (LF) und FLOAT (Freizeit).

  • Personen-Datenbank Online Link zu Personen-Datenbank Online

    Diese Webanwendung ermöglicht die bequeme Verwaltung von Personendaten in einer Online-Datenbank. Nutzer können Einträge hinzufügen, löschen, bearbeiten, sortieren und filtern. Die Anwendung basiert auf Thymeleaf, Spring Boot und Java und bietet eine intuitive Benutzeroberfläche für die effiziente Verwaltung von Personaldaten.

  • Personen-Datenbank Konsole Link zu Personen-Datenbank Konsole

    Diese Java-Konsolenanwendung ermöglicht die Verwaltung von Personendaten in einer lokalen Datenbank. Nutzer können Einträge hinzufügen, löschen, bearbeiten, sortieren und filtern. Die Anwendung bietet eine intuitive Benutzeroberfläche über die Befehlszeile für die effiziente Verwaltung von Personaldaten.

  • Tic Tac Toe Link zu Tic Tac Toe

    Tic Tac Toe ist ein klassisches Spiel für zwei Spieler, bei dem es darum geht, als erster eine Reihe von drei Symbolen (entweder X oder O) in einem 3x3-Spielfeld zu platzieren. Dieses Java-Programm ermöglicht es Ihnen, Tic Tac Toe in der Konsole gegen einen Freund zu spielen.
